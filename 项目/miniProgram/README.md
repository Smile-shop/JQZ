## 云展厅-微信小程序
云展厅-微信小程序是基于微信小程序开发的一款小程序。用于展示首饰和饰品(由后台配置)。依赖的库有，有赞 vant-weapp(UI库)、wxParser-plugin(解析富文本)。

## 启动
使用微信开发者工具打开云展厅-微信小程序
